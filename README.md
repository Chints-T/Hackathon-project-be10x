<h3> 1. Project Overview </h3>
Pearl Dental Studio Clinic is a website built for a dental clinic based in Koramangala, Bangalore, India. It serves as a complete digital presence for the clinic — combining a professional marketing website, an online appointment booking system, a blog, and a 24/7 AI voice receptionist (Juhi) — all within a single downloadable file.

<h3> Who Is This For? </h3>

Patients — finding treatment info, pricing, booking appointments, getting directions
Clinic Staff — showcasing doctors, services, and collecting appointment leads
Clinic Management — building brand trust, reducing phone call load via AI voice agent
